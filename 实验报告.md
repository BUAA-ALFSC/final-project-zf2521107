# 实验报告：基于元学习的 PINN 损失函数发现方法

## 1. 实验目的

本实验旨在验证基于 **First-Order MAML 元学习算法**自动发现物理信息神经网络（PINN）最优损失权重的有效性。具体目标包括：

1. 复现 Psaros 等人 (2022) 提出的元学习 PINN 损失函数方法
2. 实现三种损失参数化：**标量权重** (2参数)、**LAL自适应损失** (4参数)、**FFN损失网络** (1720参数)
3. 在一维 Burgers 方程上验证元学习是否能发现优于 MSE/L1/Cauchy 基线的自适应损失函数
4. 评估学习到的损失函数在**未见过的 PDE 参数**上的泛化能力

## 3. 验证问题与任务设计

### 3.1 一维 Burgers 方程

$$u_t + u u_x = \nu u_{xx}, \quad x \in [-1, 1], t \in [0, 1]$$

- **初始条件**：$u(x, 0) = -\sin(\pi x)$
- **边界条件**：$u(-1, t) = u(1, t) = 0$
- **关键特性**：非线性对流项 $u \cdot u_x$ 导致激波形成，对 PINN 训练构成挑战

### 3.2 元学习任务分布

通过改变粘度系数 $\nu$ 构造任务分布 $p(\nu)$：

| 任务集 | 粘度系数 $\nu$ | 数量 | 用途 |
|--------|----------------|------|------|
| 训练任务 | $0.005/\pi$, $0.008/\pi$, $0.01/\pi$, $0.015/\pi$, $0.02/\pi$ | 5 | 元训练（学习损失权重） |
| 测试任务 | $0.012/\pi$ | 1 | 元测试（评估泛化能力） |

**设计原理**：测试任务粘度位于训练任务区间中间，严格检验内插泛化能力。

### 3.3 数据配置

每个任务的数据集构成：

| 数据类型 | 点数 | 说明 |
|----------|------|------|
| 边界条件点 | 200 (每侧) | $x = \pm 1$，$u = 0$ |
| 初始条件点 | 400 | $t = 0$，$u = -\sin(\pi x)$ |
| 内部配点 | 2000 | 随机采样，计算 PDE 残差 |
| 训练/验证分割 | 80%/20% | 随机划分 |

### 3.4 参考解

使用 **Method of Lines + 迎风格式 + RK45自适应步长** 生成精确参考解：

- 空间离散：迎风格式（对流项）+ 中心差分（扩散项）
- 时间积分：SciPy `solve_ivp` 的 RK45 自适应步长方法
- 自动满足稳定性约束，无需手动 CFL 检查
- 初始条件误差 < $10^{-16}$，边界条件误差 < $10^{-16}$

| 参数 | 初始值 | 说明 |
|------|--------|------|
| $\alpha_u, \alpha_f$ | 2.01 | 形状参数（$\alpha \to 2$ 趋近 MSE） |

| 可学习参数数 | 4 | $(\alpha_u, \log c_u, \alpha_f, \log c_f)$ |

### 4.2.2 FFN 损失网络参数化（论文 Section 3.4.1.2）

| 参数 | 值 | 说明 |
|------|-----|------|
| 网络结构 | [2, 40, 40, 1] | 输入 $(u_{pred}, u_{true})$，输出逐点损失值 |
| 激活函数 | ReLU + ReLU + Softplus | Softplus 保证非负损失输出 |
| 偏置 | 无 | 所有层不使用偏置 |
| 初始化 | Xavier 均匀 | $\text{limit} = \sqrt{6/(\text{fan\_in} + \text{fan\_out})}$ |
| 总参数量 | 1720 | $W_1(40\times 2) + W_2(40\times 40) + W_3(1\times 40)$ |
| 预训练 | 200 步 Adam (lr=1e-3) | 在合成 MSE 数据上预训练，初始化近似 MSE |
| 正则化 | Eq.28 (coef=0.01, s=0.01) | 最优平稳条件 + MSE 关系条件 |
| 外循环学习率 | 1e-4 | Adam 优化器（FFN 参数多，学习率更低） |
| 元训练轮数 | 500 | 计算资源限制（论文使用 10000） |

| 超参数 | 值 | 说明 |

| 外循环最小学习率 $\beta_{\min}$ | $10^{-3}$ | 余弦退火下限 |
| 损失权重范围 | [0.05, 20.1] | $\lambda = \exp(w)$, $w \in [-3, 3]$ |

### 4.3 元测试超参数

| 超参数 | 值 |
|--------|-----|
| 优化器 | Adam |
| 学习率 | $10^{-4}$ |
| 训练轮数 | 1000 |

### 4.4 学习率调度策略

外循环采用**余弦退火**（Cosine Annealing）：

$$\beta_t = \beta_{\min} + \frac{1}{2}(\beta_0 - \beta_{\min})\left(1 + \cos\left(\frac{\pi t}{M}\right)\right)$$

其中 $M = 200$ 为总迭代数。前期大学习率快速探索，后期小学习率精细收敛。

## 5. 实验流程

### 5.1 第一阶段：数据准备与验证

1. 对每个训练/测试任务的 $\nu$ 值，使用 FD 方法生成参考解
2. 验证初始条件和边界条件的数值精度
3. 生成 BC/IC 点和内部配点的训练/验证分割

### 5.2 第二阶段：元训练

对每一轮元训练迭代：

1. **内循环**（每个任务独立执行）：
   - 从基模型参数 $\theta$ 克隆
   - 使用当前损失权重 $(w_u, w_f)$ 执行 $K=5$ 步 SGD 适配
   - 适配目标：最小化该任务训练集上的 PINN 损失

2. **外循环**：
   - 对适配后参数应用 `stop_gradient`（First-Order MAML 近似）
   - 在所有任务验证集上计算元损失
   - 累积损失权重梯度并求平均
   - 使用余弦退火学习率更新 $(w_u, w_f)$
   - 对损失权重施加软约束：$w \in [-3, 3]$

3. **基模型更新**：
   - 在所有任务训练数据上计算基模型损失
   - 使用固定学习率 $10^{-4}$ 更新基模型参数 $\theta$

### 5.3 第三阶段：元测试

1. 固定元训练学习到的损失权重 $\lambda_u^* = \exp(w_u^*)$, $\lambda_f^* = \exp(w_f^*)$
2. 在未见过的测试任务 ($\nu = 0.012/\pi$) 上**从头训练**两个 PINN：
   - **方案 A**：使用元学习权重
   - **方案 B**：使用均匀权重 ($\lambda_u = \lambda_f = 1$)
3. 比较 L2 相对误差、数据损失、PDE 残差损失

## 6. 实验结果

### 6.1 LAL 元训练收敛

- 元损失在 1000 轮迭代中从 0.287 下降至 0.072（外循环 Adam 有效优化）
- LAL 参数演化：
  - $\alpha_u$: 2.01 → 1.77（从 MSE-like 趋向 L1-like），$c_u$: 0.71 → 1.40
  - $\alpha_f$: 2.01 → 1.98（保持 MSE-like），$c_f$: 0.71 → 1.43
- 与标量权重方法一致，LAL 降低数据项对 MSE 的依赖（$\alpha_u$ 偏离 2），保持 PDE 项接近 MSE

### 6.2 FFN 元训练收敛

- FFN 预训练：200 步 Adam，MSE 从 13.09 降至 1.21，成功初始化接近 MSE 的损失函数形状
- 元损失在 500 轮迭代中从 1.7855 下降至 1.4265（外循环 Adam lr=1e-4）
- FFN 权重 RMS 范数演化：
  - $|W_1|$: 0.0477 → 0.0437（40×2 输入层权重收敛）
  - $|W_2|$: 0.0301 → 0.0285（40×40 隐藏层权重收缩）
  - $|W_3|$: 0.0675 → 0.0706（1×40 输出层权重略增）
- Eq.28 正则化项周期性非零（如 Iter 0: 3.07, Iter 100: 2.45, Iter 200: 1.22），表明 FFN 逐渐偏离 MSE 形状，但正则化将偏离控制在一定范围内
- 与 LAL 类似，FFN 也学习到偏重 PDE 约束的损失函数形状

### 6.3 标量权重演化（前版本）

- $\lambda_{data}$（BC/IC 权重）：从 1.0 逐渐下降至 0.772
- $\lambda_{PDE}$（PDE 残差权重）：基本稳定在 1.0 附近（最终 0.9995）

两种参数化方法一致收敛至偏重 PDE 残差的策略，验证了发现的稳健性。

### 6.4 元测试：FFN + LAL vs 多基线

在测试任务 ($\nu = 0.012/\pi$) 上的定量结果：

| 方法 | L2 相对误差 | 数据损失 (MSE) | PDE 残差 (MSE) |
|------|-----------|---------------|---------------|
| FFN (ours) | 0.999 | 2.56×10⁻¹ | **2.17×10⁻⁵** |
| LAL (ours) | 0.710 | 9.64×10⁻² | 3.67×10⁻² |
| MSE | **0.532** | **4.23×10⁻²** | 8.57×10⁻² |
| L1 | 0.794 | 1.49×10⁻¹ | 4.14×10⁻³ |
| Cauchy | 0.647 | 8.13×10⁻² | 3.37×10⁻² |

### 6.5 标量权重元测试（前版本）

| 指标 | 元学习权重 | 均匀权重 (基线) | 变化 |
|------|-----------|----------------|------|
| L2 相对误差 | 0.691 | 0.544 | +27.1% |
| 数据损失 (BC/IC) | 9.35×10⁻² | 4.80×10⁻² | +94.7% |
| PDE 残差损失 | 3.22×10⁻² | 7.24×10⁻² | **-55.5%** |

> 具体数值因随机种子略有波动。三种参数化方法（FFN、LAL、标量权重）均一致地降低 PDE 残差（FFN: -99.97%, LAL: -49%, 标量: -55%），但数据拟合精度下降导致整体 L2 误差未超越 MSE 基线。FFN（1720 参数）表现最极端，几乎完全忽略数据拟合（L2=0.999）。L1 也极端偏重 PDE 约束，Cauchy 表现最接近 MSE。

## 8. 讨论与分析

### 8.1 First-Order MAML 近似的有效性

实验表明，通过 `stop_gradient` 断开适配参数的二阶梯度路径是有效的。该方法：
- 将计算复杂度从四阶梯度降至一阶梯度
- 保留了元学习的核心能力——学习跨任务通用的损失权重
- 在 MindSpore 的自动微分框架下实现简洁

### 8.2 损失函数泛化机制

元学习学到的不是针对特定 $\nu$ 的损失函数，而是**跨任务通用的数据-物理平衡策略**。三种参数化方法一致地发现降低数据拟合重要性的策略：

- **FFN**（1720 参数）：PDE 残差降低 99.97%（PDE=2.17×10⁻⁵），但数据拟合完全退化（L2=0.999）
- **标量权重**：$\lambda_u \to 0.77$，PDE 残差降低 55.5%
- **LAL**：$\alpha_u \to 1.77$（趋向 L1），PDE 残差降低 49.3%

这种跨参数化的一致性强烈表明：在 Burgers 方程上，偏重物理约束是跨任务通用的有效策略。然而，参数容量越高，策略越极端——FFN 的 1720 个参数给予了损失函数过大的灵活性，导致在有限训练任务（5 个）和有限元训练轮数（500 轮）下过拟合。Eq.28 正则化（coef=0.01）在当前设置下不足以约束 FFN 保持在合理范围内。

L1 损失和 FFN 的极端结果说明过度强调 PDE 约束会导致数据拟合严重退化——存在一个最优平衡点，当前的元学习设置倾向于过度偏向 PDE 一侧。未来可尝试：(1) 增大 Eq.28 正则化系数或增加训练任务数以遏制 FFN 过拟合；(2) 在验证损失中更好地平衡数据和 PDE 项；(3) 增加元训练迭代数至论文的 10000 轮。

### 8.3 数值稳定性设计

- **MOL-RK45 参考解**：替代 Cole-Hopf 变换，避免小 $\nu$ 时的数值不稳定问题；迎风格式有效处理激波区域；RK45 自适应步长自动满足稳定性约束
- **损失权重软约束**：通过 $\tanh$ 软钳制限制 $\lambda$ 范围，防止梯度爆炸
- **梯度函数预定义**：在模块级别定义 `ms.grad`，避免重复编译开销

### 8.4 局限性

1. 仅在一维 Burgers 方程上验证，需扩展到更多 PDE 类型
2. FFN 参数化已实现（1720 参数，500 轮元训练），但受限于计算资源，元训练不充分（论文使用 10000 轮）
3. FFN 在当前设置下严重过拟合（L2=0.999），Eq.28 正则化强度需要进一步调优
4. 元测试中整体 L2 误差未超越 MSE 基线，三种参数化方法均未达到预期
5. 仅 5 个训练任务，500-1000 轮元训练（论文使用 10000），统计显著性和收敛充分性有限
6. 内循环使用 SGD 而非 Adam，但元测试使用 Adam——优化器不匹配可能影响泛化
7. 三种方法均未超越 MSE，说明 Burgers 方程上 MSE 可能接近最优，或当前实验设置不足以发挥元学习优势

## 9. 结论

本实验成功使用 MindSpore 框架复现了 Psaros 等人 (2022) 的元学习 PINN 损失函数方法，实现了从简单标量权重到 LAL 自适应损失再到 FFN 损失网络的完整演进。主要结论：

1. **跨参数化一致性**：三种参数化方法（标量权重、LAL、FFN）一致地发现偏重 PDE 残差的策略（PDE 损失降低 49-99.97%），验证了元学习发现的稳健性
2. **参数容量效应**：参数容量越高（FFN 1720 > LAL 4 > 标量 2），学习到的策略越极端偏重 PDE，揭示了过参数化在有限训练数据下的过拟合风险
3. **LAL 有效性**：LAL 参数化（4参数）成功实现论文 Section 3.4.1.1，自动满足最优平稳条件和 MSE 关系条件，无需额外正则化
4. **FFN 实现**：成功实现论文 Section 3.4.1.2 的 FFN 损失网络（1720 参数，Eq.28 正则化），但受限于训练规模和正则化强度，出现严重过拟合（L2=0.999）
5. **多基线对比**：MSE 在当前设置下表现最优（L2=0.532），Cauchy（0.647）是最接近的替代方案，L1（0.794）和 FFN（0.999）说明极端偏重 PDE 不利整体精度
6. **稳定性**：通过 MOL-RK45 参考解、FFN 预训练、Xavier 初始化、梯度预定义等设计保证了数值稳定

该方法为减少 PINN 手动调参负担、实现自动化损失函数发现提供了可行途径。未来工作将聚焦于 FFN 正则化强度调优、更长元训练（10000 轮）和更广泛 PDE 验证以进一步发挥方法的潜力。
